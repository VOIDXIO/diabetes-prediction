# Diabetes Prediction System

A machine learning project that predicts the likelihood of diabetes using patient health data. The project demonstrates the end-to-end process of building a predictive model, including data preprocessing, exploratory data analysis, feature engineering, model training, and evaluation.

## 📌 Project Overview

Diabetes is a chronic disease that affects millions of people worldwide. Early prediction can help individuals seek timely medical attention and adopt preventive measures. This project applies machine learning techniques to analyze medical diagnostic data and predict whether a person is likely to have diabetes.

## 📂 Project Structure

```
├── diabetes_prediction.ipynb    # Jupyter Notebook containing the complete ML workflow
├── dataset.csv                  # Dataset used for training and evaluation
└── README.md                    # Project documentation
```

## 📊 Dataset Information

The dataset contains medical diagnostic measurements used to predict diabetes. The target variable, **Outcome**, indicates whether an individual has diabetes (`1`) or not (`0`).

### Features
- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI (Body Mass Index)
- Diabetes Pedigree Function
- Age
- Outcome (Target Variable)
